# TableServe

A casual dining order management system built as a reference application for code academy.


## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/craigmckeachie/tableserve.git
cd tableserve
```

### 2. Remove the existing Git history

```bash
# Mac/Linux
rm -rf .git

# Windows (Command Prompt)
rmdir /s /q .git

# Windows (PowerShell)
Remove-Item -Recurse -Force .git
```
